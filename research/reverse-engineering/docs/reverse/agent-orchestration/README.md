## 🎓 关键洞察

从 Claude Code 逆向中提炼的核心洞察：

1. **上下文管理 > 模型能力** - Anthropic 最大的投资在上下文管理
2. **React 组件即 Loop** - 声明式模型天然适合 Agent Loop
3. **权限即管道** - 权限检查嵌入执行链，而非前置门卫
4. **文件系统即数据库** - 简单、可审计、可手动修复
5. **Hook 注入点是扩展边界** - 5 层 Hook 实现完整可扩展性
6. **渐进式自动化** - Manual → Plan → Auto → Speculation
7. **错误类型即领域语言** - 完整的错误分类体系
