# 踩坑记录

> 遇到问题和解决方案都记录在这里，避免重复踩坑

## 飞书 lark-cli

### 1. Cookie 过期问题
- **问题**: Cookie 会在几小时后过期，导致 API 调用失败
- **解决**: 实现 `lark setup` 命令快速重新配置
- **教训**: 需要实现自动刷新机制

### 2. Protobuf 解码顺序
- **问题**: 直接用 protobuf 库解码会丢失未知字段
- **解决**: 先用 protobuf-inspector 盲解码，再逐步构建 schema
- **教训**: 永远先盲解码，再定义 schema

### 3. WebSocket 认证
- **问题**: WebSocket 连接后需要发送特定的认证帧
- **解决**: 分析抓包数据，还原认证帧格式
- **教训**: WebSocket 连接后的前几帧通常是认证相关的
