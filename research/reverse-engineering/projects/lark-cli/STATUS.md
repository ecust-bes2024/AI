# lark-cli 项目状态

## 当前状态

- **阶段**: 基础功能已完成，进入增强和扩展阶段
- **版本**: v0.1.0（非正式）

## 已完成功能

- [x] 用户认证（Cookie 方式）
- [x] 消息发送（按用户名/chat_id）
- [x] 联系人搜索
- [x] 实时消息监听（WebSocket）
- [x] Protobuf 协议解析
- [x] Claude Code 集成（/lark 命令）

## 技术架构

- **认证**: Cookie + CSRF Token
- **协议**: Protobuf over HTTP
- **实时通信**: WebSocket (wss://msg-frontier.feishu.cn/ws/v2)
- **技术栈**: Python 3 + requests + protobuf + websockets

## 技术债务

- proto_pb2.py 是自动生成的，需要保留原始 .proto 文件
- WebSocket 重连机制需要优化
- Cookie 过期检测和自动刷新
- 配置文件格式需要标准化（考虑支持 TOML）

## 风险和限制

- 使用用户凭据，非官方 API
- Cookie 会过期，需要定期更新
- 可能触发反爬虫检测
- 协议可能随飞书更新而变化
